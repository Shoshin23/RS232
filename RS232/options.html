If you wish to only use this as a prediction tool then dont add your MtGox account details just yet. The app uses the first traded price of each hour for the EMA input.
A (too) simple explanation of how this app trades.

if the value on the Badge of the extension is above the threshold value it indicates a buy.
If the value on the Badge is below the threshold value it sells.
Reading more on crossovers and EMAs would shed light on how this works + the Threshold concept was a little addition by me to decrease the amount of false signals that this technique could trigger if a volatile market like Bitcoins.
